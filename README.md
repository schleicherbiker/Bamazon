# Bamazon
Amazon-like Node Application Using MySQL

## When you run bamazonCustomer.js, you will be presented with all the items for sale.

![alt text](https://github.com/schleicherbiker/Bamazon/raw/master/Images/ItemList.PNG "Items")

## Type the number or item ID to select an item.

![alt text](https://github.com/schleicherbiker/Bamazon/raw/master/Images/ItemList_2.PNG "Items")

## Once an item is selected, enter the quantity you would like to buy.

![alt text](https://github.com/schleicherbiker/Bamazon/raw/master/Images/BuyItem.PNG "Items")

## After a valid quantity is entered, the total price will be logged. The quantity change is reflected in both the console and the database.

![alt text](https://github.com/schleicherbiker/Bamazon/raw/master/Images/BuyItem_2.PNG "Items")

## In the case that the quantity entered is greater than the quantity in stock, the user will be notified that there is insufficient stock.

![alt text](https://github.com/schleicherbiker/Bamazon/raw/master/Images/Quantity.PNG "Items")
